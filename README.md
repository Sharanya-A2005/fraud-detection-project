Fraud Detection System

Problem Statement:

Detect fraudulent online transactions using machine learning techniques.

Solution:

This system predicts whether a transaction is fraudulent based on:

* Amount 
* Location 
* Time 
* Transaction Type 

Technologies Used:

* Python (Machine Learning)
* Flask (API)
* Node.js (Backend)
* MongoDB (Database)
* HTML, CSS, JavaScript (Frontend)

Workflow:

1. User enters transaction details
2. Data is sent to Node.js backend
3. Backend calls Flask ML API
4. Model predicts fraud
5. Result shown in UI

How to Run

Step 1: Train Model

python model.py

Step 2: Start ML API

python predict.py

Step 3: Start Backend

node server.js

Step 4: Open Frontend

Open index.html in browser
